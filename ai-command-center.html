GUÍA — CONFIGURAR ESTA APP EN TU VM
📋 PASOS PARA TRABAJAR DESDE EL DASHBOARD EN TU VM DE CLOUD CLUSTERS
1
SUBIR A GITHUB
Crear repo ai-command-center en GitHub. Subir este archivo HTML. Conectar a Vercel. URL quedará como ai-command-center.vercel.app
2
ABRIR DESDE TU VM
Dentro de Cloud Clusters, abrir Chrome y entrar a la URL de Vercel. Hacer bookmark en la barra. Desde ahí controlas todo.
3
PONER TUS API KEYS
En la pestaña MIS IAs, clic en Editar Key de cada IA. Las keys se guardan en tu Google Drive en una carpeta privada llamada OC-001-MARCO/api-keys.json
4
CONECTAR CLOUDING.IO
Poner tu API key de Clouding.io en las variables de entorno de Vercel. Con eso los botones ⛔ Apagar VM y ➕ Nueva VM funcionan de verdad via API.
5
MIGRAR A VM ESTABLE (cuando proyecto madure)
